#########################
How to install:
#########################

1. Copy directory "TAZ" directly to "C:/".
2. Add "C:/TAZ/ICONS" to your Support File Search Path (in AutoCAD).
3. Restart AutoCAD.
4. Type in your command line (in AutoCAD): (load "C:\\TAZ\\_INSTALL.lsp")

#########################
How to uninstall:
#########################

1. Type in your command line (in AutoCAD): (load "C:\\TAZ\\_UNINSTALL.lsp")
